# 캐릭터 표시 함수 (`Character`) 사용 안내서

안녕하세요! 이 문서는 `Character` 함수를 사용해서 화면에 캐릭터 이미지를 원하는 위치에 표시하는 방법을 알려주는 안내서입니다.

## 사용법 요약

1. **이미지 준비:** 배경이 투명한 PNG 이미지를 준비합니다.
2. **폴더에 추가:** `app` > `src` > `main` > `res` > `drawable` 폴더에 이미지를 넣습니다.
3. **코드 작성:** `Character(가로, 세로, 크기, 명도, "이미지이름")` 형식으로 사용합니다.

--- 

## 단계별 상세 설명

### 1단계: 캐릭터 이미지 준비하기

배경이 투명한 PNG 이미지를 준비합니다. (예: `hero.png`, `villain.png`)

### 2단계: 프로젝트에 이미지 추가하기

안드로이드 스튜디오에서 `app` > `src` > `main` > `res` > `drawable` 폴더에 이미지를 드래그 앤 드롭합니다.

### 3단계: 코드로 캐릭터 표시하기

`Character` 함수는 5개의 파라미터를 순서대로 입력합니다:

```kotlin
Character(가로위치, 세로위치, 크기, 명도, "이미지이름")
```

**파라미터 설명:**
1. **가로 위치** (1~10): 1=왼쪽, 5=중앙, 10=오른쪽
2. **세로 위치** (1~10): 1=위, 5=중앙, 10=아래
3. **크기**: 이미지 크기 (50~500 권장)
4. **명도** (0~200): 0=어둡게, 100=보통, 200=밝게
5. **이미지 이름**: drawable 폴더의 파일명 (확장자 제외)

**사용 예시:**

```kotlin
// 화면 중앙에 기본 설정 (크기 200, 밝기 100)
Character(5, 5, 200, 100, "hero")

// 왼쪽 위에 작게
Character(1, 1, 150, 100, "hero")

// 오른쪽 아래에 크고 어둡게
Character(10, 10, 300, 50, "villain")

// 중앙에 밝게
Character(5, 5, 200, 150, "angel")
```

### 가로 위치(Horizontal Position) 가이드

`horizontalPosition` 값에 따라 캐릭터의 가로 위치가 달라집니다:

| Position | 위치 설명 |
|----------|-----------|
| 1 | 가장 왼쪽 |
| 2 | 왼쪽 |
| 3 | 중앙-왼쪽 |
| 4 | 약간 왼쪽 |
| 5 | 중앙-왼쪽 |
| 6 | 중앙-오른쪽 |
| 7 | 약간 오른쪽 |
| 8 | 중앙-오른쪽 |
| 9 | 오른쪽 |
| 10 | 가장 오른쪽 |

### 세로 위치(Vertical Position) 가이드

`verticalPosition` 값에 따라 캐릭터의 세로 위치가 달라집니다:

| Position | 위치 설명 |
|----------|-----------|
| 1 | 가장 위 |
| 2 | 위 |
| 3 | 중앙-위 |
| 4 | 약간 위 |
| 5 | 중앙-위 |
| 6 | 중앙-아래 |
| 7 | 약간 아래 |
| 8 | 중앙-아래 |
| 9 | 아래 |
| 10 | 가장 아래 |

### 기본 사용 예시

```kotlin
// 중앙에 hero 캐릭터를 기본 설정으로 표시
Character(
    horizontalPosition = 5,
    verticalPosition = 5,
    imageName = "hero"
)
```

## 크기 가이드

| 크기 | 설명 |
|------|------|
| 50~100 | 작은 캐릭터 |
| 150~250 | 일반 캐릭터 (기본 200) |
| 300~400 | 큰 캐릭터 |
| 500 이상 | 매우 큰 캐릭터 |

## 명도 가이드

| 명도 | 효과 |
|------|------|
| 0 | 완전 어둡게 (검은 실루엣) |
| 50 | 매우 어둡게 |
| 100 | 보통 (기본값) |
| 150 | 밝게 |
| 200 | 매우 밝게 |

## 주의사항

⚠️ **이미지 이름 규칙:**
- 소문자, 숫자, 밑줄(_)만 사용
- 대문자나 특수문자 사용 불가
- 확장자(.png) 제외

**좋은 예:** `hero.png` → `"hero"` ✅  
**나쁜 예:** `My-Hero.png` → `"My-Hero"` ❌
