# 선택지 함수 (`Choice`) 사용 안내서

안녕하세요! 이 문서는 `Choice` 함수를 사용해서 사용자에게 두 가지 선택지를 제시하고, 선택 결과를 받는 방법을 알려주는 안내서입니다.

## 사용법 요약

1.  **함수 호출:** `Choice` 함수를 호출합니다.
2.  **선택지 입력:** 두 개의 버튼에 표시할 텍스트를 입력합니다.
3.  **결과 처리:** 사용자가 선택한 버튼 번호(1 또는 2)를 받아서 원하는 동작을 실행합니다.

--- 

## 단계별 상세 설명

### 1단계: Choice 함수 이해하기

`Choice` 함수는 화면에 두 개의 버튼을 가로로 나란히 표시하는 함수입니다.

-   **디자인 특징:**
    -   역사적이고 고풍스러운 느낌의 갈색 톤
    -   금색 테두리로 고급스러운 분위기
    -   화면 중앙보다 약간 아래에 배치
    -   두 버튼이 가로로 나란히 배치

-   **동작 방식:**
    -   사용자가 왼쪽 버튼을 클릭하면 → `1`을 반환
    -   사용자가 오른쪽 버튼을 클릭하면 → `2`를 반환

### 2단계: 코드 작성하기

아래 예시 코드처럼 `Choice` 함수를 사용합니다.

**기본 예시 코드:**

```kotlin
// 화면을 구성하는 Composable 함수 안에서...

Choice(
    option1 = "예",           // 왼쪽 버튼에 표시할 텍스트
    option2 = "아니오"        // 오른쪽 버튼에 표시할 텍스트
) { choice ->
    // choice 값이 1이면 "예" 버튼을 클릭한 것입니다.
    // choice 값이 2이면 "아니오" 버튼을 클릭한 것입니다.
    
    when (choice) {
        1 -> println("사용자가 '예'를 선택했습니다!")
        2 -> println("사용자가 '아니오'를 선택했습니다!")
    }
}
```

### 3단계: 다양한 활용 예시

#### 예시 1: 게임에서 선택지 표시

```kotlin
Choice(
    option1 = "공격하기",
    option2 = "도망가기"
) { choice ->
    if (choice == 1) {
        // 공격 시작
        startBattle()
    } else {
        // 도망
        runAway()
    }
}
```

#### 예시 2: 설정 확인 대화상자

```kotlin
Choice(
    option1 = "저장",
    option2 = "취소"
) { choice ->
    when (choice) {
        1 -> {
            saveSettings()
            showToast("설정이 저장되었습니다")
        }
        2 -> {
            cancelChanges()
            showToast("취소되었습니다")
        }
    }
}
```

#### 예시 3: 역사적 선택 상황

```kotlin
Choice(
    option1 = "왕과 협상하기",
    option2 = "전쟁을 선포하기"
) { choice ->
    if (choice == 1) {
        negotiateWithKing()
        showMessage("평화로운 길을 선택했습니다")
    } else {
        declareWar()
        showMessage("전쟁의 길로 들어섰습니다")
    }
}
```

### 4단계: 함수 파라미터 설명

| 파라미터 | 타입 | 설명 |
|---------|------|------|
| `option1` | String | 왼쪽(첫 번째) 버튼에 표시될 텍스트 |
| `option2` | String | 오른쪽(두 번째) 버튼에 표시될 텍스트 |
| `onChoice` | (Int) -> Unit | 사용자가 선택한 버튼 번호(1 또는 2)를 받는 콜백 함수 |

### 5단계: 주의사항

-   **반환 값:** 첫 번째 버튼은 `1`, 두 번째 버튼은 `2`를 반환합니다.
-   **텍스트 길이:** 너무 긴 텍스트는 버튼에 다 표시되지 않을 수 있으니, 짧고 명확한 텍스트를 사용하세요.
-   **전체 화면:** `Choice` 함수는 화면 전체를 사용하므로, 단독으로 사용하는 것이 좋습니다.

## 디자인 커스터마이징

현재 `Choice` 함수는 고풍스러운 역사 테마로 디자인되어 있습니다:

-   **1번 버튼:** 밝은 갈색 배경 (`#8B4513`)
-   **2번 버튼:** 어두운 갈색 배경 (`#654321`)
-   **테두리:** 금색 (`#D4AF37`)
-   **텍스트:** 베이지색 (`#F5DEB3`)

디자인을 변경하고 싶다면, 함수 내부의 `Color` 값들을 수정하면 됩니다.
