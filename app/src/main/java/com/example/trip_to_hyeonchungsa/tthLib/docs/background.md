# 배경화면 바꾸기 함수 (`SetBackground`) 사용 안내서

안녕하세요! 이 문서는 `SetBackground` 함수를 사용해서 앱의 배경화면을 원하는 이미지로 바꾸는 방법을 알려주는 안내서입니다.

## 사용법 요약

1.  **이미지 준비:** 배경으로 쓰고 싶은 이미지 파일을 준비합니다.
2.  **폴더에 추가:** 준비한 이미지 파일을 안드로이드 스튜디오의 특정 폴더에 넣습니다.
3.  **코드 수정:** `SetBackground` 함수를 코드에 추가하고, 이미지 파일 이름을 알려줍니다.

--- 

## 단계별 상세 설명

### 1단계: 배경 이미지 준비하기

먼저, 배경으로 사용하고 싶은 이미지(예: `my_cat.jpg` 또는 `beautiful_sky.png`)를 컴퓨터에 준비해주세요.

### 2단계: 프로젝트에 이미지 추가하기

1.  안드로이드 스튜디오 왼쪽의 프로젝트 탐색기에서 `app` > `src` > `main` > `res` > `drawable` 폴더를 찾아주세요.
2.  준비한 이미지 파일을 이 `drawable` 폴더 안으로 드래그 앤 드롭해서 복사해 넣습니다.

    *이제 프로젝트에서 이 이미지를 사용할 수 있게 되었습니다!*

### 3단계: 코드로 배경 설정하기

이제 코드에서 `SetBackground` 함수를 불러와 배경을 설정할 차례입니다.

아래 예시 코드처럼 `SetBackground` 함수를 사용하고, `imageName` 부분에 `drawable` 폴더에 넣은 **이미지 파일의 이름 (확장자 제외)**을 정확히 적어주세요.

**예시 코드:**

만약 `drawable` 폴더에 `my_cat.png` 파일을 넣었다면, 아래와 같이 코드를 작성합니다.

```kotlin
// 화면을 구성하는 Composable 함수 안에서...

// "my_cat" 이라는 이름의 이미지로 배경을 설정합니다.
SetBackground(imageName = "my_cat") {
    
    // 이 안에는 배경 위에 보여줄 내용을 넣습니다.
    // 예를 들어, "안녕!" 이라는 글자를 보여줄 수 있습니다.
    Text("안녕!")

}
```
