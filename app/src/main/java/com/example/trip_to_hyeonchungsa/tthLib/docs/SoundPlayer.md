# 음악 재생 함수 (SoundPlayer) 사용 안내서

안녕하세요! 이 문서는 `SoundPlayer` 함수를 사용해서 앱에서 음악이나 효과음을 재생하는 방법을 알려주는 안내서입니다.

## 사용법 요약

1. **음악 파일 준비:** 재생하고 싶은 음악 파일을 준비합니다.
2. **폴더에 추가:** 준비한 음악 파일을 안드로이드 스튜디오의 특정 폴더에 넣습니다.
3. **코드 수정:** `SoundPlayer` 함수를 코드에 추가하고, 음악 파일 이름과 설정을 알려줍니다.

---

## 단계별 상세 설명

### 1단계: 음악 파일 준비하기

먼저, 재생하고 싶은 음악 파일(예: `background_music.mp3` 또는 `click_sound.wav`)을 컴퓨터에 준비해주세요.

### 2단계: 프로젝트에 음악 파일 추가하기

1. 안드로이드 스튜디오 왼쪽의 프로젝트 탐색기에서 `app` > `src` > `main` > `res` > `raw` 폴더를 찾아주세요.
   - 만약 `raw` 폴더가 없다면, `res` 폴더에서 마우스 우클릭 > **New > Directory**를 선택하고 `raw`라고 입력해서 새로 만들어주세요.
2. 준비한 음악 파일을 이 `raw` 폴더 안으로 드래그 앤 드롭해서 복사해 넣습니다.

*이제 프로젝트에서 이 음악 파일을 사용할 수 있게 되었습니다!*

### 3단계: 코드로 음악 재생하기

이제 코드에서 `SoundPlayer` 함수를 불러와 음악을 재생할 차례입니다.

아래 예시 코드처럼 `SoundPlayer` 함수를 사용하고, 각 설정값을 입력해주세요.

**예시 코드:**

만약 `raw` 폴더에 `background_music.mp3` 파일을 넣었다면, 아래와 같이 코드를 작성합니다.

```kotlin
// 화면을 구성하는 Composable 함수 안에서...

// "background_music" 음악을 50% 볼륨으로, 반복하지 않고 재생합니다.
SoundPlayer("background_music", 50, false) {
    
    // 이 안에는 음악이 재생되는 동안 보여줄 내용을 넣습니다.
    // 예를 들어, "음악이 재생됩니다!" 라는 글자를 보여줄 수 있습니다.
    Text("음악이 재생됩니다!")

}
```

**파라미터 설명:**
- **첫 번째 ("background_music")**: 음악 파일 이름 (확장자 `.mp3` 제외)
- **두 번째 (50)**: 소리 크기 (1~100, 50은 절반 크기)
- **세 번째 (false)**: 무한 반복 여부 (`true`면 반복, `false`면 한 번만 재생)
- **중괄호 { } 안**: 음악이 재생되는 동안 화면에 보여줄 내용

---

## 음악 제어 방법

음악은 `SoundPlayer`가 화면에 있을 때만 재생됩니다. 

- **음악 재생**: `SoundPlayer`를 화면에 표시
- **음악 정지**: `SoundPlayer`를 화면에서 제거

**예시:**

```kotlin
@Composable
fun MyScreen() {
    var showMusic by remember { mutableStateOf(true) }
    
    // showMusic이 true일 때만 음악 재생
    if (showMusic) {
        SoundPlayer("background_music", 50, true) {
            Text("음악 재생 중...")
        }
    }
}
```

**설명:** `showMusic` 값을 변경하면 `SoundPlayer`가 화면에서 사라지면서 자동으로 음악이 멈춥니다.
